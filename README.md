# Smart Campus Energy Monitoring System

This project is a web-based solution to monitor and analyze campus energy consumption in real time.

## Features
- Real-time energy data storage using Firebase
- Interactive dashboard with tables and graphs
- Building-wise filtering and alerts
- Deployed using Firebase Hosting (Google Cloud)

## Tech Stack
- HTML, CSS, JavaScript
- Firebase Realtime Database
- Chart.js
- Google Cloud Platform

## How to Run
1. Open `dashboard/index.html` in a browser
2. Ensure Firebase database is accessible for read

## Use Case
Helps campus administrators identify energy wastage and optimize consumption.
